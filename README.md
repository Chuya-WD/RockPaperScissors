# RockPaperScissors
Este juego se jugará íntegramente en la consola.

# Fase 1
1. Creación del repositorio RockPaperScissors en Github
2. Creación del documento .html con la estructura básica y una etiqueta script para el js externo

3. vincular el documento .html con el documento .js y verificar la correcta vinculación con console.log("hola mundo");


# Fase 2 (Requisitos)

Este juego se jugara contra la computadora. 

La computadora debe devolver piedra papel o tijera por medio de una función getComputerChoice()

La función getComputerChoice() retornara uno de estos valores al azar: piedra, papel o tijera. Se debe usar la función Math.random().
