# Fase 2 (Requisitos)

Este juego se jugara contra la computadora. 

La computadora debe devolver piedra papel o tijera por medio de una función getComputerChoice()

La función getComputerChoice() retornara uno de estos valores al azar: piedra, papel o tijera. Se debe usar la función Math.random(). 


El juego debe pedir al usuario que escriba su elección (piedra, papel o tijeras) y devolver ese valor mediante una función getHumanChoice()


El juego debera almacenar la puntuacción de los jugadores en variables humanScore y computerScore

El juego debe decirdir el ganador en base a las elecciones de los 
jugadores. Se debe implementarlo mediante una función donde se actualice los puntos e imprima un mensaje.
